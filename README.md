# BiologicalDataSharing
This is a repository where we stored our files used in the course Biological Data Sharing, taught at Universiteit Leiden


# File structure
For the files we separated the 2 folders based on the phylogeny method. 
- sequence_files, holds both files used for the sequence-based phylogeny.
- structure_files, holds all the .cif files used in the structure_based phylogeny.

## sequence_files
- sequences_unaligned.FASTA, is the file where we stored all protein sequences before use. The sequences were only cleaned such that it was easy to see which species belongs to which sequence.
- sequences_aligned_Clustal.fa, is the output file we recieved from Clustal Omega in Fasta format.

## structure_files
- This directory holds all .cif files which were used in the Foldmason MSA structure-based phylogeny.
